1. 介绍

ncurses(new curses)是一套编程库，它提供了一系列的函数以便使用者调用它们去生成基于文本的用户界面。
ncurses 名字中的n意味着“new”，因为它是curses的自由软件版本。由于 AT&T“臭名昭著”的版权政策，人们不得不在后来用
ncurses 去代替它。
ncurses 是GNU 计划的一部分，但它却是少数几个不使用GNU GPL 或 LGPL 授权的 GNU 软件之一。

其实我们对 ncurses 本身并不陌生，以下几款大名鼎鼎的软件都用到过 ncurses：

  • vim
  • emacs
  • lynx
  • screen
