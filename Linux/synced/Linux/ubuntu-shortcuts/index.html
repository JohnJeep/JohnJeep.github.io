Ubuntu 桌面快捷键

按键 描述
Win+Tab 快速切换窗口，只显示应用图标
Win+A 搜索/浏览程序
Win+Alt+← 将当前窗口左移一个显示器
Win+Alt+→ 将当前窗口右移一个显示器
Win+V 显示通知列表
Ctrl + Alt + 方向键 在不同的工作区之间切换
