й«ҳе№¶еҸ‘

еӨҡиҝӣзЁӢе№¶еҸ‘жңҚеҠЎеҷЁ

е®һзҺ°зҡ„жҖқжғіпјҡз”ЁзҲ¶еӯҗиҝӣзЁӢ(fork)е’ҢдҝЎеҸ·жҚ•жҚү(signal()жҲ–sigaction)еҺ»е®һзҺ°гҖӮзҲ¶иҝӣзЁӢе…ій—ӯе®ўжҲ·з«Ҝзҡ„ж–Үд»¶жҸҸиҝ°з¬ҰпјҢе№¶иҝӣиЎҢдҝЎеҸ·зҡ„жҚ•жҚүпјҢеӣһж”¶еӯҗиҝӣзЁӢпјҢжқҘеӨ„зҗҶеғөе°ёиҝӣзЁӢпјӣеӯҗиҝӣзЁӢе…ій—ӯжңҚеҠЎз«Ҝзҡ„ж–Үд»¶жҸҸиҝ°з¬ҰпјҢиҝӣиЎҢдҝЎеҸ·зҡ„жіЁеҶҢгҖӮ

еӨҡзәҝзЁӢе№¶еҸ‘жңҚеҠЎеҷЁ

  • е®һзҺ°зҡ„жҖқжғіпјҡи®ҫзҪ®зәҝзЁӢзҡ„еұһжҖ§е’ҢзәҝзЁӢзҡ„еҲҶзҰ» pthread_detach
  • ж— и·ҜжҳҜйҮҮз”ЁеӨҡиҝӣзЁӢе№¶еҸ‘жңҚеҠЎеҷЁиҝҳжҳҜеӨҡзәҝзЁӢе№¶еҸ‘жңҚеҠЎеҷЁпјҢеҜ№жңҚеҠЎеҷЁзҡ„дә§з”ҹзҡ„ејҖй”ҖйғҪжҜ”иҫғеӨ§пјҢдёҖиҲ¬еҸӘз”ЁдёҖдәӣжҜ”иҫғе°Ҹзҡ„еңәеҗҲпјҢеғҸеӨ§еһӢзҡ„зҪ‘з»ңдёҖиҲ¬дёҚйҖӮз”ЁпјҢйңҖиҰҒз”ЁеҲ°еӨҡи·ҜIOиҪ¬жҺҘжЁЎеһӢжқҘе®һзҺ°гҖӮ

еӨҡи·Ҝ IO еӨҚз”Ё

UDP е’Ң TCP жңҖеҹәжң¬зҡ„иҙЈд»»жҳҜ: е°ҶдёӨдёӘз«Ҝзі»з»ҹй—ҙ IP зҡ„дј йҖ’жңҚеҠЎжү©еұ•дёәиҝҗиЎҢеңЁз«Ҝзі»з»ҹдёҠзҡ„дёӨдёӘиҝӣзЁӢд№Ӣй—ҙзҡ„дј йҖ’жңҚеҠЎгҖӮе°Ҷдё»жңәй—ҙдј йҖ’жү©еұ•еҲ°иҝӣзЁӢй—ҙдј йҖ’иў«з§°дёәдј иҫ“еұӮзҡ„еӨҡи·ҜеӨҚз”Ё (transport-layer multiplexing) дёҺеӨҡи·ҜеҲҶи§Ј (demultiplexing)

  • еӨҡи·ҜIOиҪ¬жҺҘжЁЎеһӢеҺҹзҗҶпјҡдёҚеҶҚз”ұеә”з”ЁзЁӢеәҸ(жңҚеҠЎеҷЁ)зӣҙжҺҘзӣ‘еҗ¬е®ўжҲ·з«ҜпјҢиҖҢжҳҜйҖҡиҝҮеҶ…ж ёжӣҝд»Јеә”з”ЁзЁӢеәҸиҝӣиЎҢзӣ‘еҗ¬ж–Үд»¶гҖӮ
  • жҹҘзңӢдёҖдёӘиҝӣзЁӢеҸҜд»Ҙжү“ејҖsockж–Үд»¶жҸҸиҝ°з¬Ұзҡ„дёҠйҷҗеҖј cat /proc/sys/fs/file-max
  • з«ҜеҸЈеӨҚз”ЁеҮҪж•°
    • setsockopt() :дёҖиҲ¬жҸ’е…ҘеңЁ socket() еҮҪж•°дёҺ bind() еҮҪж•°д№Ӣй—ҙгҖӮ

select

select е’Ң poll йғҪжҳҜ UNIX/Linux зі»з»ҹдёӯзҡ„ I/O еӨҡи·ҜеӨҚз”ЁжңәеҲ¶пјҢз”ЁдәҺеҗҢж—¶зӣ‘жҺ§еӨҡдёӘж–Үд»¶жҸҸиҝ°з¬Ұзҡ„зҠ¶жҖҒеҸҳеҢ–гҖӮ

еҹәжң¬е·ҘдҪңжңәеҲ¶

  1. ж–Үд»¶жҸҸиҝ°з¬ҰйӣҶеҗҲпјҡ

    • дҪҝз”ЁдёүдёӘдҪҚжҺ©з Ғ(fd_set)еҲҶеҲ«иЎЁзӨәreadгҖҒwrite е’ҢејӮеёёдәӢд»¶(expect)
    • жҜҸдёӘ bit д»ЈиЎЁдёҖдёӘж–Үд»¶жҸҸиҝ°з¬Ұзҡ„зҠ¶жҖҒ

  2. е·ҘдҪңжөҒзЁӢпјҡ

    • еә”з”ЁзЁӢеәҸеҲқе§ӢеҢ– fd_set йӣҶеҗҲпјҢи®ҫзҪ®иҰҒзӣ‘жҺ§зҡ„ж–Үд»¶жҸҸиҝ°з¬Ұ
    • и°ғз”Ёselect()зі»з»ҹи°ғз”Ё
    • еҶ…ж ёзәҝжҖ§жү«жҸҸжүҖжңүиў«зӣ‘жҺ§зҡ„ж–Үд»¶жҸҸиҝ°з¬Ұ
    • еҪ“жңүдәӢд»¶еҸ‘з”ҹжҲ–и¶…ж—¶еҗҺпјҢselectиҝ”еӣһ
    • еә”з”ЁзЁӢеәҸйңҖиҰҒйҒҚеҺҶжүҖжңүж–Үд»¶жҸҸиҝ°з¬ҰжүҫеҮәе°ұз»Әзҡ„

  3. еҮҪж•°еҺҹеһӢ

зү№зӮ№дёҺйҷҗеҲ¶

  1. ж–Үд»¶жҸҸиҝ°з¬Ұж•°йҮҸйҷҗеҲ¶пјҡ
    • йҖҡеёёз”ұFD_SETSIZE е®ҡд№ү(йҖҡеёёжҳҜ1024)
    • ж— жі•й«ҳж•ҲеӨ„зҗҶеӨ§йҮҸиҝһжҺҘ
  2. жҖ§иғҪй—®йўҳпјҡ
    • жҜҸж¬Ўи°ғз”ЁйғҪйңҖиҰҒе°Ҷж•ҙдёӘfd_setд»Һз”ЁжҲ·з©әй—ҙжӢ·иҙқеҲ°еҶ…ж ёз©әй—ҙ
    • иҝ”еӣһж—¶йңҖиҰҒеҶҚж¬ЎжӢ·иҙқдҝ®ж”№еҗҺзҡ„fd_setеӣһз”ЁжҲ·з©әй—ҙ
    • жҜҸж¬Ўи°ғз”ЁйғҪйңҖиҰҒзәҝжҖ§жү«жҸҸжүҖжңүж–Үд»¶жҸҸиҝ°з¬Ұ(O(n)еӨҚжқӮеәҰ)
  3. йҮҚеӨҚеҲқе§ӢеҢ–пјҡ
    • selectдјҡдҝ®ж”№дј е…Ҙзҡ„ fd_setпјҢеӣ жӯӨжҜҸж¬Ўи°ғз”ЁеүҚйғҪйңҖиҰҒйҮҚж–°еҲқе§ӢеҢ–
  4. зјәзӮ№
    • Linuxдёӯ select зӣ‘еҗ¬ж–Үд»¶жҸҸиҝ°з¬Ұзҡ„зҡ„жңҖеӨ§еҖјдёә 1024
    • йңҖиҰҒиҮӘе®ҡдёҖдёӘж•°жҚ®з»“жһ„(ж•°з»„)еҺ»йҒҚеҺҶе“Әдәӣж–Үд»¶жҸҸиҝ°з¬Ұж»Ўи¶іжқЎд»¶гҖӮ
    • жҜҸж¬ЎиҝӣиЎҢж“ҚдҪңзҡ„ж—¶еҖҷпјҢйңҖиҰҒе°Ҷзӣ‘еҗ¬зҡ„йӣҶеҗҲе’Ңж»Ўи¶іжқЎд»¶зӣ‘еҗ¬зҡ„йӣҶеҗҲиҝӣиЎҢдҝқеӯҳпјҢеӣ дёәжҜҸж¬Ўзҡ„ж“ҚдҪңдјҡдҝ®ж”№еҺҹжңүйӣҶеҗҲзҡ„еҖјгҖӮ
  5. еӣӣдёӘиҫ…еҠ©еҮҪж•°
  • void FD_ZERO(fd_set *set) е°Ҷsetжё…0
  • void FD_CLR(int fd, fd_set *set) е°Ҷfdд»Һsetдёӯжё…йҷӨеҮәеҺ»
  • void FD_SET(int fd, fd_set *set) е°Ҷfdи®ҫзҪ®еҲ°setйӣҶеҗҲдёӯеҺ»
  • int FD_ISSET(int fd, fd_set *set) еҲӨж–ӯfdжҳҜеҗҰеңЁsetйӣҶеҗҲдёӯ

poll

еҹәжң¬е·ҘдҪңжңәеҲ¶

  1. ж–Үд»¶жҸҸиҝ°з¬ҰиЎЁзӨәпјҡ

    дҪҝз”Ё pollfd з»“жһ„дҪ“ж•°з»„пјҢжҜҸдёӘе…ғзҙ еҢ…еҗ«пјҡ

  2. е·ҘдҪңжөҒзЁӢпјҡ

    • еә”з”ЁзЁӢеәҸеҲқе§ӢеҢ– pollfd ж•°з»„
    • и°ғз”Ё poll() зі»з»ҹи°ғз”Ё
    • еҶ…ж ёзәҝжҖ§жү«жҸҸжүҖжңүиў«зӣ‘жҺ§зҡ„ж–Үд»¶жҸҸиҝ°з¬Ұ
    • еҪ“жңүдәӢд»¶еҸ‘з”ҹжҲ–и¶…ж—¶еҗҺпјҢpoll иҝ”еӣһ
    • еә”з”ЁзЁӢеәҸйңҖиҰҒйҒҚеҺҶ pollfd ж•°з»„жЈҖжҹҘ revents еӯ—ж®ө

  3. еҮҪж•°еҺҹеһӢпјҡ

    • fds ж•°з»„зҡ„йҰ–ең°еқҖ
    • nfds ж•°з»„дёӯе…ғзҙ зҡ„дёӘж•°
    • timeout и¶…ж—¶ж—¶й—ҙ(еҚ•дҪҚдёәmsзә§еҲ«)
      • -1 йҳ»еЎһзӯүеҫ…
      • 0 з«ӢеҚіиҝ”еӣһпјҢдёҚйҳ»еЎһзӯүеҫ…
      • >0 зӯүеҫ…жҢҮе®ҡзҡ„ж—¶й—ҙй•ҝеәҰ

зү№зӮ№дёҺж”№иҝӣ

  1. ж— ж–Үд»¶жҸҸиҝ°з¬Ұж•°йҮҸйҷҗеҲ¶пјҡ
    • зҗҶи®әдёҠеҸӘеҸ—зі»з»ҹиө„жәҗйҷҗеҲ¶
    • еҸҜд»ҘеӨ„зҗҶжҜ”selectжӣҙеӨҡзҡ„иҝһжҺҘ
  2. жӣҙзҒөжҙ»зҡ„дәӢд»¶е®ҡд№үпјҡ
    • еҸҜд»Ҙзӣ‘жҺ§жӣҙеӨҡзұ»еһӢзҡ„дәӢд»¶
    • дәӢд»¶е’Ңиҝ”еӣһз»“жһңеҲҶзҰ»(eventsе’Ңrevents)
  3. жҖ§иғҪй—®йўҳпјҡ
    • д»Қз„¶йңҖиҰҒжҜҸж¬Ўи°ғз”Ёж—¶дј йҖ’ж•ҙдёӘз»“жһ„дҪ“ж•°з»„
    • д»Қз„¶йңҖиҰҒзәҝжҖ§жү«жҸҸжүҖжңүж–Үд»¶жҸҸиҝ°з¬Ұ(O(n)еӨҚжқӮеәҰ)
    • дҪҶйҒҝе…ҚдәҶselectзҡ„ fd_set йҮҚеӨҚеҲқе§ӢеҢ–й—®йўҳ

дјҳзӮ№

  • зӣ‘еҗ¬ж–Үд»¶жҸҸиҝ°з¬Ұзҡ„иҝ”еӣһеҖјеҸҜд»Ҙи¶…иҝҮ 1024
  • е®һзҺ°дәҶзӣ‘еҗ¬йӣҶеҗҲдёҺиҝ”еӣһйӣҶеҗҲзҡ„еҲҶзҰ»
  • д»…еңЁж•°з»„дёӯжҗңзҙўпјҢиҢғеӣҙеҸҳе°ҸдәҶпјҢдҪҶжҳҜж•ҲзҺҮиҝҳжҳҜжҜ”иҫғдҪҺ

epoll

epoll жҳҜLinuxзі»з»ҹдёӯдёҖз§Қй«ҳж•Ҳзҡ„I/OдәӢд»¶йҖҡзҹҘжңәеҲ¶пјҢе®ғжҳҜ select/poll зҡ„ж”№иҝӣзүҲжң¬пјҢиғҪеӨҹеӨ„зҗҶеӨ§йҮҸж–Үд»¶жҸҸиҝ°з¬Ұзҡ„I/OдәӢд»¶пјҢеёёз”ЁдәҺжһ„е»әй«ҳжҖ§иғҪзҡ„зҪ‘з»ңжңҚеҠЎеҷЁгҖӮ

ж ёеҝғеҺҹзҗҶ

  1. дәӢд»¶й©ұеҠЁжЁЎеһӢпјҡ

    • epoll йҮҮз”ЁдәӢд»¶й©ұеҠЁзҡ„ж–№ејҸпјҢеҸӘжңүеҪ“ж–Үд»¶жҸҸиҝ°з¬ҰзҠ¶жҖҒеҸ‘з”ҹеҸҳеҢ–ж—¶жүҚдјҡйҖҡзҹҘеә”з”ЁзЁӢеәҸ
    • йҒҝе…ҚдәҶselect/poll йңҖиҰҒиҪ®иҜўжүҖжңүж–Үд»¶жҸҸиҝ°з¬Ұзҡ„ејҖй”Җ

  2. дёүдёӘе…ій”®зі»з»ҹи°ғз”Ёпјҡ

    • epoll_create()пјҡеҲӣе»әдёҖдёӘepollе®һдҫӢпјҢиҝ”еӣһж–Үд»¶жҸҸиҝ°з¬Ұ

    • epoll_ctl()пјҡеҗ‘epollе®һдҫӢжіЁеҶҢгҖҒдҝ®ж”№жҲ–еҲ йҷӨж„ҹе…ҙи¶Јзҡ„ж–Үд»¶жҸҸиҝ°з¬Ұе’ҢдәӢд»¶

    • epoll_wait()пјҡзӯүеҫ…I/OдәӢд»¶зҡ„еҸ‘з”ҹпјҢиҝ”еӣһе°ұз»Әзҡ„ж–Үд»¶жҸҸиҝ°з¬Ұ

е·ҘдҪңжөҒзЁӢ

  1. еҲқе§ӢеҢ–йҳ¶ж®өпјҡ
    • и°ғз”Ёepoll_create()еҲӣе»әepollе®һдҫӢ
    • йҖҡиҝҮepoll_ctl()жіЁеҶҢйңҖиҰҒзӣ‘жҺ§зҡ„ж–Үд»¶жҸҸиҝ°з¬Ұе’ҢдәӢд»¶зұ»еһӢ(иҜ»/еҶҷ/й”ҷиҜҜзӯү)
  2. дәӢд»¶зӯүеҫ…йҳ¶ж®өпјҡ
    • и°ғз”Ёepoll_wait()йҳ»еЎһзӯүеҫ…дәӢд»¶еҸ‘з”ҹ
    • еҶ…ж ёйҖҡиҝҮеӣһи°ғжңәеҲ¶зӣ‘жҺ§жіЁеҶҢзҡ„ж–Үд»¶жҸҸиҝ°з¬Ұ
  3. дәӢд»¶йҖҡзҹҘйҳ¶ж®өпјҡ
    • еҪ“жҹҗдёӘж–Үд»¶жҸҸиҝ°з¬Ұе°ұз»Әж—¶пјҢеҶ…ж ёе°Ҷе…¶еҠ е…Ҙе°ұз»ӘеҲ—иЎЁ
    • epoll_wait()иҝ”еӣһе°ұз»Әзҡ„ж–Үд»¶жҸҸиҝ°з¬Ұе’ҢеҜ№еә”зҡ„дәӢд»¶
  4. дәӢд»¶еӨ„зҗҶйҳ¶ж®өпјҡ
    • еә”з”ЁзЁӢеәҸеӨ„зҗҶе°ұз»Әзҡ„ж–Үд»¶жҸҸиҝ°з¬Ұ
    • еӨ„зҗҶе®ҢжҜ•еҗҺз»§з»ӯи°ғз”Ёepoll_wait()зӯүеҫ…ж–°дәӢд»¶

жҖ§иғҪдјҳеҠҝ

  1. й«ҳж•Ҳзҡ„ж•°жҚ®з»“жһ„пјҡ
    • дҪҝз”Ёзәўй»‘ж ‘еӯҳеӮЁжүҖжңүеҫ…зӣ‘жҺ§зҡ„ж–Үд»¶жҸҸиҝ°з¬ҰпјҢжҹҘжүҫж•ҲзҺҮй«ҳ(O(log n))
    • е°ұз»ӘеҲ—иЎЁдҪҝз”ЁеҸҢеҗ‘й“ҫиЎЁе®һзҺ°пјҢдәӢд»¶еҸ‘з”ҹж—¶еҸӘйңҖе°Ҷе°ұз»ӘйЎ№еҠ е…Ҙй“ҫиЎЁ
  2. иҫ№зјҳи§ҰеҸ‘(ET)е’Ңж°ҙе№іи§ҰеҸ‘(LT)жЁЎејҸпјҡ
    • иҫ№зјҳи§ҰеҸ‘(ET)пјҡеҸӘеңЁзҠ¶жҖҒеҸҳеҢ–ж—¶йҖҡзҹҘдёҖж¬Ў
    • ж°ҙе№іи§ҰеҸ‘(LT)пјҡеҸӘиҰҒж»Ўи¶іжқЎд»¶е°ұжҢҒз»ӯйҖҡзҹҘ
    • ETжЁЎејҸиғҪеҮҸе°‘йҮҚеӨҚйҖҡзҹҘпјҢжҸҗй«ҳж•ҲзҺҮ
  3. йҒҝе…ҚдёҚеҝ…иҰҒзҡ„жӢ·иҙқпјҡ
    • жҜҸж¬Ўи°ғз”Ёepoll_wait()ж—¶дёҚйңҖиҰҒеғҸselect/pollйӮЈж ·еҗ‘еҶ…ж ёдј йҖ’жүҖжңүж–Үд»¶жҸҸиҝ°з¬Ұ
    • еҶ…ж ёз»ҙжҠӨзқҖжҢҒд№…зҡ„дәӢд»¶жіЁеҶҢдҝЎжҒҜ

epoll зү№еҲ«йҖӮеҗҲеӨ„зҗҶеӨ§йҮҸе№¶еҸ‘иҝһжҺҘдҪҶжҙ»еҠЁиҝһжҺҘжҜ”дҫӢдёҚй«ҳзҡ„еңәжҷҜпјҢеҰӮпјҡ

  • WebжңҚеҠЎеҷЁ
  • еҚіж—¶йҖҡи®ҜжңҚеҠЎеҷЁ
  • жёёжҲҸжңҚеҠЎеҷЁ
  • е…¶д»–й«ҳе№¶еҸ‘зҪ‘з»ңеә”з”Ё

epoll зҡ„й«ҳж•ҲжҖ§дҪҝе…¶жҲҗдёәLinuxдёӢејҖеҸ‘й«ҳжҖ§иғҪзҪ‘з»ңжңҚеҠЎеҷЁзҡ„йҰ–йҖүI/OеӨҡи·ҜеӨҚз”ЁжңәеҲ¶гҖӮ

epollеҸҚеә”е ҶжЁЎеһӢ

  • ж ёеҝғжҖқжғіпјҡи°ғз”ЁдәҶ libevent еә“
  • дҪҝз”Ё libevent еә“дјҳзӮ№пјҡеә“зҡ„еә•еұӮеӨ§йҮҸйҮҮз”ЁдәҶеӣһи°ғзҡ„жҖқжғіпјҢеҚіеҮҪж•°жҢҮй’Ҳзҡ„дҪҝз”ЁпјҢеҗҢж—¶д№ҹжҳҜи·Ёе№іеҸ°зҡ„гҖӮ

дёүз§Қи§ҰеҸ‘жЁЎејҸ

иҫ№зјҳи§ҰеҸ‘(Edge-Triggered, ET)е’Ңж°ҙе№іи§ҰеҸ‘(Level-Triggered, LT)жҳҜI/OеӨҡи·ҜеӨҚз”ЁжңәеҲ¶дёӯдёӨз§ҚдёҚеҗҢзҡ„дәӢд»¶йҖҡзҹҘж–№ејҸпјҢе®ғд»¬еҶіе®ҡдәҶж“ҚдҪңзі»з»ҹеҰӮдҪ•йҖҡзҹҘеә”з”ЁзЁӢеәҸж–Үд»¶жҸҸиҝ°з¬Ұзҡ„зҠ¶жҖҒеҸҳеҢ–гҖӮ

epoll ET

е·ҘдҪңеҺҹзҗҶпјҡ

  • д»…еңЁж–Үд»¶жҸҸиҝ°з¬ҰзҠ¶жҖҒеҸ‘з”ҹеҸҳеҢ–ж—¶йҖҡзҹҘдёҖж¬Ў(еҰӮд»ҺдёҚеҸҜиҜ»еҸҳдёәеҸҜиҜ»пјҢжҲ–д»ҺдёҚеҸҜеҶҷеҸҳдёәеҸҜеҶҷ)
  • еҚідҪҝзј“еҶІеҢәдёӯд»Қжңүж•°жҚ®пјҢеҰӮжһңжІЎжңүж–°зҡ„ж•°жҚ®еҲ°иҫҫпјҢдёҚдјҡеҶҚж¬ЎйҖҡзҹҘ

зү№зӮ№пјҡ

  1. зұ»дјјдәҺвҖқзҠ¶жҖҒеҸҳеҢ–ж—¶еҚ•ж¬ЎйҖҡзҹҘвҖқзҡ„жЁЎејҸ
  2. еә”з”ЁзЁӢеәҸеҝ…йЎ»дёҖж¬ЎжҖ§еӨ„зҗҶе®ҢжүҖжңүж•°жҚ®(еҫӘзҺҜиҜ»еҸ–зӣҙеҲ° EAGAIN/EWOULDBLOCK)
  3. иғҪеҮҸе°‘йҮҚеӨҚйҖҡзҹҘпјҢжҸҗй«ҳж•ҲзҺҮ
  4. зј–зЁӢеӨҚжқӮеәҰиҫғй«ҳпјҢе®№жҳ“йҒ—жјҸдәӢд»¶

зӨәдҫӢеңәжҷҜпјҡ

  • еҪ“socketжҺҘж”¶зј“еҶІеҢәд»Һз©әеҸҳдёәйқһз©әж—¶пјҢеҸӘйҖҡзҹҘдёҖж¬ЎеҸҜиҜ»дәӢд»¶пјҢеҚідҪҝд№ӢеҗҺзј“еҶІеҢәд»ҚжңүжңӘиҜ»ж•°жҚ®д№ҹдёҚдјҡеҶҚйҖҡзҹҘ

жҖ»з»“

  • еҸӘжңүclientеҸ‘йҖҒж•°жҚ®ж—¶пјҢжүҚдјҡи§ҰеҸ‘гҖӮ
  • и°ғз”Ёпјҡevent = EPOLLIN | EPOLLET
  • clientе°ҶеӨ§йҮҸзҡ„ж•°жҚ®еӯҳеҲ°epollзҡ„зј“еҶІеҢәдёӯж—¶пјҢserverеҸӘд»ҺepollдёӯиҜ»еҸ–дёҖйғЁеҲҶзҡ„ж•°жҚ®пјҢиҝҷйғЁеҲҶзҡ„ж•°жҚ®жҰӮжӢ¬дәҶзј“еҶІеҢәдёӯжүҖжңүж•°жҚ®зҡ„дҝЎжҒҜпјҢзј“еҶІеҢәдёӯеү©дҪҷзҡ„ж•°жҚ®ж №жҚ®йңҖжұӮжқҘиҝӣиЎҢеҸ–иҲҚгҖӮ

epoll LT

е·ҘдҪңеҺҹзҗҶпјҡ

  • еҸӘиҰҒж–Үд»¶жҸҸиҝ°з¬ҰеӨ„дәҺе°ұз»ӘзҠ¶жҖҒ(иҜ»зј“еҶІеҢәжңүж•°жҚ®еҸҜиҜ»/еҶҷзј“еҶІеҢәжңүз©әй—Із©әй—ҙ)пјҢе°ұдјҡжҢҒз»ӯйҖҡзҹҘеә”з”ЁзЁӢеәҸ
  • еә”з”ЁзЁӢеәҸеҸҜд»ҘдёҚз«ӢеҚіеӨ„зҗҶе®ҢжүҖжңүж•°жҚ®пјҢдёӢж¬Ўи°ғз”Ё epoll_wait() ж—¶дјҡеҶҚж¬ЎйҖҡзҹҘ

зү№зӮ№пјҡ

  1. зұ»дјјдәҺвҖқжқЎд»¶жҢҒз»ӯж»Ўи¶іе°ұжҢҒз»ӯйҖҡзҹҘвҖқзҡ„жЁЎејҸ
  2. еҰӮжһңдәӢд»¶жңӘиў«е®Ңе…ЁеӨ„зҗҶпјҢдёӢж¬ЎжЈҖжҹҘж—¶дјҡеҶҚж¬ЎжҠҘе‘Ҡ
  3. зј–зЁӢжЁЎеһӢзӣёеҜ№з®ҖеҚ•пјҢдёҚе®№жҳ“йҒ—жјҸдәӢд»¶
  4. еҸҜиғҪйҖ жҲҗдёҚеҝ…иҰҒзҡ„йҮҚеӨҚйҖҡзҹҘ
  5. зі»з»ҹдёҚеҒҡд»»дҪ•зҡ„еЈ°жҳҺпјҢдёҖиҲ¬й»ҳи®ӨжҳҜж°ҙе№іи§ҰеҸ‘гҖӮ
  6. дёҖж¬ЎжҖ§еҜ№ read() еҮҪж•°иҝӣиЎҢж“ҚдҪңпјҢеҸӘжңү read() еҮҪж•°жү§иЎҢе®ҢеҗҺпјҢжүҚдјҡиҝӣиЎҢж°ҙе№іи§ҰеҸ‘гҖӮ

зӨәдҫӢеңәжҷҜпјҡ

  • еҪ“socketжҺҘж”¶зј“еҶІеҢәжңүж•°жҚ®ж—¶пјҢжҜҸж¬Ўepoll_wait()йғҪдјҡжҠҘе‘ҠеҸҜиҜ»дәӢд»¶пјҢзӣҙеҲ°зј“еҶІеҢәиў«иҜ»з©ә

йқһйҳ»еЎһIOж–№ејҸ

  • дјҳзӮ№пјҡеҮҸе°‘ epoll_wait еҮҪж•°и°ғз”Ёзҡ„ж¬Ўж•°пјҢжҸҗй«ҳж•ҲзҺҮгҖӮ
  • open() еҮҪж•°пјҢеңЁsocketеҘ—жҺҘеӯ—дёӯдёҚйҖӮз”ЁгҖӮ
  • з»“еҗҲ fcntl() еҮҪж•°е’Ң readn() еҮҪж•°дёҖиө·дҪҝз”ЁгҖӮ readn() дёҖж¬ЎжҖ§иҜ»еҸ– n дёӘеӯ—иҠӮеҗҺжүҚиҝ”еӣһгҖӮ
  • еҰӮдҪ•дҪҝз”Ёпјҹ
    • дҪҝз”Ёиҫ№жІҝи§ҰеҸ‘ж–№ејҸ
    • жү§иЎҢиҝҮзЁӢдёӯдҪҝз”Ё while(read())
    • и°ғз”Ё fcntl(0_NOBLOCK)

select poll epollеҜ№жҜ”

  1. select:
    • жңҖеҸӨиҖҒзҡ„I/OеӨҡи·ҜеӨҚз”ЁжңәеҲ¶
    • жңүж–Үд»¶жҸҸиҝ°з¬Ұж•°йҮҸйҷҗеҲ¶(FD_SETSIZE)
    • жҜҸж¬Ўи°ғз”ЁйңҖиҰҒйҮҚж–°и®ҫзҪ®ж–Үд»¶жҸҸиҝ°з¬ҰйӣҶеҗҲ
    • зәҝжҖ§жү«жҸҸжүҖжңүж–Үд»¶жҸҸиҝ°з¬ҰпјҢйҡҸзқҖзӣ‘жҺ§ж•°йҮҸеўһеҠ пјҢжҖ§иғҪжҳҺжҳҫдёӢйҷҚ
    • жҜҸж¬Ўи°ғз”ЁйғҪйңҖиҰҒеңЁз”ЁжҲ·з©әй—ҙе’ҢеҶ…ж ёз©әй—ҙд№Ӣй—ҙжӢ·иҙқзӣ‘жҺ§еҲ—иЎЁ
    • еҸӘж”ҜжҢҒж°ҙе№іи§ҰеҸ‘(LT)жЁЎејҸ
  2. poll:
    • и§ЈеҶідәҶselectзҡ„ж–Үд»¶жҸҸиҝ°з¬Ұж•°йҮҸйҷҗеҲ¶
    • дҪҝз”Ёpollfdз»“жһ„дҪ“ж•°з»„пјҢжӣҙзҒөжҙ»
    • д»Қз„¶йңҖиҰҒзәҝжҖ§жү«жҸҸжүҖжңүж–Үд»¶жҸҸиҝ°з¬ҰпјҢйҡҸзқҖзӣ‘жҺ§ж•°йҮҸеўһеҠ пјҢжҖ§иғҪжҳҺжҳҫдёӢйҷҚ
    • жҜҸж¬Ўи°ғз”ЁйғҪйңҖиҰҒеңЁз”ЁжҲ·з©әй—ҙе’ҢеҶ…ж ёз©әй—ҙд№Ӣй—ҙжӢ·иҙқзӣ‘жҺ§еҲ—иЎЁ
    • дёҚйңҖиҰҒжҜҸж¬ЎйҮҚж–°еҲқе§ӢеҢ–ж•°жҚ®з»“жһ„
    • еҸӘж”ҜжҢҒж°ҙе№іи§ҰеҸ‘(LT)жЁЎејҸ
  3. epoll:
    • Linuxзү№жңүпјҢжҖ§иғҪжңҖеҘҪ
    • дҪҝз”Ёзәўй»‘ж ‘еӯҳеӮЁзӣ‘жҺ§зҡ„ж–Үд»¶жҸҸиҝ°з¬ҰпјҢж•ҲзҺҮй«ҳ
    • ж”ҜжҢҒиҫ№зјҳи§ҰеҸ‘(ET)е’Ңж°ҙе№іи§ҰеҸ‘(LT)жЁЎејҸ
    • еҸӘиҝ”еӣһе°ұз»Әзҡ„ж–Үд»¶жҸҸиҝ°з¬ҰпјҢдёҚйңҖиҰҒзәҝжҖ§жү«жҸҸ
    • йҖӮеҗҲй«ҳе№¶еҸ‘еңәжҷҜ

е…¶е®ғж–№йқўеҜ№жҜ”

  1. ж•°жҚ®з»“жһ„пјҡ

    • select/pollпјҡжҜҸж¬Ўи°ғз”Ёдј йҖ’е®Ңж•ҙеҲ—иЎЁ

    • epollпјҡеҶ…ж ёз»ҙжҠӨжҢҒд№…зҡ„ж•°жҚ®з»“жһ„

  2. ж—¶й—ҙеӨҚжқӮеәҰпјҡ

    • select/pollпјҡO(n)
    • epollпјҡO(1)еҜ№дәҺе°ұз»Әж–Үд»¶жҸҸиҝ°з¬Ұ

  3. йҖӮз”ЁеңәжҷҜпјҡ

    • select/pollпјҡйҖӮеҗҲе°‘йҮҸиҝһжҺҘжҲ–и·Ёе№іеҸ°еә”з”Ё
    • epollпјҡйҖӮеҗҲLinuxе№іеҸ°дёҠзҡ„й«ҳе№¶еҸ‘еә”з”Ё
