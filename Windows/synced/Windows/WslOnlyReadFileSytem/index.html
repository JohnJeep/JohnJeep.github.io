WSL2 文件系统只读修复手册

快速判断

出现 mounting fs with errors → 需要 e2fsck 修复。

修复步骤

第一步：确认 Ubuntu 的设备号

找到挂载在 / 的设备，记下来（本机是 sdc）。

第二步：安装 Debian（只需第一次）

第三步：用 vhd 方式挂载 Ubuntu 的磁盘

vhdx 路径按实际情况替换。

第四步：确认新出现的设备号

找到 1T、没有挂载点的那个设备（本机是 sdc）。

第五步：修复

看到 FILE SYSTEM WAS MODIFIED 说明修复成功。

第六步：重启验证
