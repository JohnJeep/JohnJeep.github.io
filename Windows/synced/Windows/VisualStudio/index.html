VisualStudio 版本对比

因为微软的版本比较乱，所以要理清版本，首先需要区分 VS 和 VC 的含义：

  • VS（Visual Studio）: 针对多语言（C++、C#、F#、J#、Asp、Web 等）的 IDE 集成开发环境
  • VC（Visual C++）: 针对 C++ 语言的 IDE 集成开发环境，也称为 MSVC

换言之，VS 包含 VC， VC 只是 VS 的其中一个工具集。
