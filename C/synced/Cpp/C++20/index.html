1. 整体概述

C++20 是继 C++11 之后最重要的标准更新，带来了革命性的变化。我们可以将主要特性分为几个核心领域：

1.1. 🎯 四大支柱特性

  1. 模块（Modules） - 替代头文件include的现代化构建方式
  2. 协程（Coroutines） - 异步编程的革命性特性
  3. 概念（Concepts） - 模板元编程的类型约束系统
  4. 范围（Ranges） - 现代化的算法和视图库

1.2. 🔧 语言核心改进

  • 三路比较运算符
  • constexpr 的极大增强
  • 初始化语句的扩展
  • 新的属性和特性

1.3. 📚 标准库增强

  • 格式化库（std::format）
  • 日期时间库
  • 跨度（span）
  • 同步库增强

现在让我们逐个深入讲解每个特性。

2. 模块（Modules）

2.1. 解决的问题

传统的 #include 机制存在的问题：

  • 文本替换，编译慢
  • 宏污染
  • 顺序依赖
  • 封装性差

2.2. 基本用法

2.3. 模块分区

2.4. 优势

  • 编译速度提升：模块只编译一次
  • 更好的封装：只有导出的符号可见
  • 无宏污染：模块内宏不影响导入者
  • 消除重复：不像头文件会被多次包含

3. 协程（Coroutines）

3.1. 核心概念

协程是可以在执行过程中暂停和恢复的函数，用于简化异步编程。

3.2. 协程关键字

  • co_await - 暂停执行直到操作完成
  • co_yield - 产生值并暂停
  • co_return - 完成协程执行
