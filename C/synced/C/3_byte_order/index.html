3 Byte Order

发表于2019-09-06|更新于2026-05-31|C
|总字数:98|阅读时长:1分钟|浏览量:

原则

  • 整数类型内部
    • 低地址存储低位，高地址存储高位
  • 栈
    • 存放局部变量
    • 先定义高地址，后定义低地址
  • 类、结构体或数组的元素
    • 先定义低地址，后定义高地址
    • 数组内存地址分配的公式：base_address + index * data_size

  • 大端小端（Big- Endian 和 Little-Endian）
